API Designer

This is the documentation of Talend API Designer’s management API.

Prerequisite: To manipulate this API, your user account must at least have the API Designer role.

Endpoint

  • https://apid.eu.cloud.talend.com/api/v1
    This is your API mock endpoint. When called, it will simulate the behavior of your API.

FormatRFC3339
name

Filter your results with the name of the API Definition

This filter allow to use a query language:

  • *: to replace 0 or more characters
  • ?: to replace 1 characters
  • \: use to escape a * or ?

Examples:

  • a*a -> return: all API definitions with a name that begins and ends with “a”
  • *a -> return: all API definitions with a name that ends with “a”
  • a* -> return: all API definitions with a name that begins with “a”
  • a-> return: all API definitions with the name equals to “a”
  • a\* -> return: all API definitions with the name equals to “a*”

SharingSet

A SharingSet is a representation of the users and groups that are allowed access to an API definition.

Important: A list of users and a list of groups must be provided, even if it may be empty.

Data constraints:

  • a SharingSet must contain a single owner (SharingLevel) and it must be a SharingUser (contained in users array)
